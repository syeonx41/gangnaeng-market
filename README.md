# 강냉마켓 🛍️  
**교내 중고물품 거래 플랫폼 (CRUD 프로젝트)**

---

## 📌 프로젝트 소개  
강냉마켓은 학교 구성원들이 사용할 수 있는 중고물품 거래 시스템입니다.  
Spring Boot와 Mustache를 기반으로 한 CRUD 중심 웹 애플리케이션입니다.

---

## ⚙️ 기술 스택  
- Java 17  
- Spring Boot 3.x  
- Mustache (서버 템플릿 엔진)  
- H2 / MySQL (선택에 따라 설정 예정)

---

## ✅ 주요 기능  
- 게시글 작성, 목록 조회, 상세 보기  
- 게시글 수정 및 삭제  
- 제목 기반 검색 기능  
- 거래 완료 상태 표시
